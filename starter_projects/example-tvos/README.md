### Dependencies
From the command line cd into /example-tvos and run:

```shell
bash get_tvos_dependencies.sh
```

### Prerequisites

You will need

- IntelliJ IDEA
- AIR 50.1+
- Xcode 16.4
- wget on macOS via `brew install wget`
- [Carthage](https://github.com/Carthage/Carthage#installing-carthage)
